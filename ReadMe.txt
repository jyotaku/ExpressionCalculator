■利用言語
  java

■利用ライブラリー
  JavaSE-1.8
  guava-19.0
  commons-lang3-3.0.jar
    
■ソース構成
  ExpressionCalculator:計算のメイン処理クラス
  ReversePolishCreator: 逆ポーランド表記への変換処理するクラス
  SubTerm:多項式を管理するクラス
  
  ExpressionCalculatorTest：テストクラス
  
■実行方法
  ExpressionClaculatorのメイン関数に引数を渡して、実行します。
  　・第１引数：入力ファイルのフルパス

■ExpressionClaculator.jar
  コンパイルした直接実行可能なJarです。
